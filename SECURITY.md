# Security Policy

## サポート対象バージョン

| バージョン   | サポート状況   |
| ------------ | -------------- |
| 最新リリース | サポート対象   |
| それ以前     | サポート対象外 |

## 脆弱性の報告

セキュリティ上の脆弱性を発見された場合は、**Issue での公開報告はお控えください。**

以下の連絡先にご報告をお願いいたします：

- **メール**: kz-chiba@pocket-queries.co.jp

### 報告に含めていただきたい情報

- 脆弱性の概要
- 再現手順
- 影響範囲の見立て
- 可能であれば修正案

### 対応の流れ

1. 報告受領後、5 営業日以内に受領確認をお送りします
2. 調査・修正を行い、修正版をリリースします
3. リリース後、報告者の了承を得た上で脆弱性情報を公開します

ご協力ありがとうございます。
